check dataCapture Server

dataCaptureが起動していなかったらStartボタンで起動してください。

check mjpg-Streamer Server

mjpg-Streamerが起動していなかったらStartボタンで起動してください。 USBカメラを付けた時や取り替えた時にもRestartが必要です。

check tty.js Server

tty.jsが起動していなかったらStartボタンで起動してください。 Shellが応答しない場合には、起動しなおしてください。

check USB devices

USBデバイスが表示されていなければ、iTelepass2を再起動してください。
