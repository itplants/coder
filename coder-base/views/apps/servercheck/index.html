check http Server

httpdが起動していなかったらStartボタンで起動してください。

check tempController

tempControllerが起動していなかったらStartボタンで起動してください。

check dataCapture Server

dataCaptureが起動していなかったらStartボタンで起動してください。

check shortPeriodControl Server

ShortPeriodScheduleが起動していなかったらStartボタンで起動してください。

check LongPeriodSchedule Server

LongPeriodScheduleが起動していなかったらStartボタンで起動してください。

check LongPeriodControl Server

LongPeriodControlが起動していなかったらStartボタンで起動してください。

check mjpg-Streamer Server

mjpg-Streamerが起動していなかったらStartボタンで起動してください。 USBカメラを付けた時や取り替えた時にもRestartが必要です。

check tty.js Server

tty.jsが起動していなかったらStartボタンで起動してください。 Shellが応答しない場合には、起動しなおしてください。

check USB devices

dataCaptureは、定期的にiTelepass03.localのセンサー情報を収集したり、指定日時にカメラの撮影を行うデーモンプロセスです。 設定ファイルが変更されると、自動的に変更を反映します。
